# React Portfolio Website

A modern personal portfolio website built using **React** with a fully functional contact form powered by **Supabase**.

## Features

- Responsive portfolio layout
- Contact form with database storage
- Messages stored in Supabase
- Automatic email notifications
- Secure environment variables

## Tech Stack

- React
- Supabase
- Vercel Serverless Functions
- Resend Email API
- CSS

## Contact Form Workflow

User submits message → Stored in Supabase → Webhook triggers → Email notification sent.
